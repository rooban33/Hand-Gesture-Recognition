Hand Gesture Recognition

Gesture control is a technology that lets you command devices through hand movements, making interaction more intuitive and hands-free in various applications like gaming, healthcare, and more.

This Project

Our project aims to revolutionize the gaming experience by implementing hand gesture recognition as a control mechanism. Using a camera , we will capture and interpret users' hand movements to perform in-game actions, such as character movement. This technology will not only provide a more immersive and interactive gaming experience but also open doors for accessible and engaging gameplay for individuals with physical disabilities, offering a unique blend of innovation and inclusivity in the gaming world.

Gesture

Gestures that can be used to rise keyboard interrupt are given below

ROCK

Can be used as UP arrow key (JUMP)

THUMBS UP

Can be used as Right arrow key (Right)

THUMBS DOWN

Can be used as LEFT arrow key (LEFT)

STOP

Can be used as DOWN arrow key (DUCK)
